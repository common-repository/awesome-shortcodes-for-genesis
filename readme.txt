This plugin will add shortcodes to Genesis 2.5 Themes. It adds most of the required functionality, so you can manage your themes correctly.

== Description ==

This plugin will enhance the functionality of Genesis 2.5 themes and add some additional functions like adding boxes, buttons, videos, code, giveaways to your post and pages.

Features:

1. Add different color boxes (with title and content) and buttons to your post and pages. 6 colors are available.
2. Boxes can be expandable, with the complete animation and header (Good for creating FAQ pages.). 
3. Add responsive videos from YouTube to the post or pages. The plugin will allow you to add videos, which will work automatically on Mobile and desktop site.
4. Insert code snippet easily from Gist.
5. Insert giveaways easily to your post and pages. Support for Rafflecopter and Punchtab.
6. Adds an easy way to insert columns. It supports two, three, four and six columns.

== Installation ==

1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin at the plugin administration page
3. Setup the plugin in the setting menu.

== Frequently Asked Questions ==

= Will it work with all the child themes of Genesis? =

Yes, It should work with every child theme of Genesis. If you face any issue, just open a support ticket.
